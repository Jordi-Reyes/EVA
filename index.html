Tutorial etiquetas basicas HTML

Body

El elemento body es el contenedor para todos los aspectos representables de un documento. Este es el lugar donde los autores arrojan todos los elementos y contenidos que sus visitantes alcanzarán. El elemento body es parte de la estructura básica de un documento.

p

El elemento p (párrafo) es el apropiado para distribuir el texto en párrafos. Sus etiquetas son: p y /p.

a

Es una etiqueta que nos ayuda a poder crear un enlace a una página web. El atributo principal de la etiqueta HTML es href, donde pondremos el enlace al que queremos conectar.

>

img

El elemento de imagen HTML img representa una imagen en el documento.

div

La etiqueta div define una división. Esta etiqueta permite agrupar varios elementos de bloque (párrafos, encabezados, listas, tablas, divisiones, etc).

style

Es el elemento encargado de indicar la información de estilo. Sus etiquetas son: style y /style (ambas obligatorias).
