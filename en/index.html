Basic HTML tags tutorial

Body

The body element is the container for all the representable aspects of a document. This is the place where the authors throw all the elements and content that their visitors will reach. The body element is part of the structure basic of a document.

p

The element p (paragraph) is appropriate to distribute the text in paragraphs. Its labels are: p and / p.

a

It is a tag that helps us to create a link to a web page. The main attribute of the HTML tag is href, where we will put the link we want to connect to.

>

img

The HTML image element img represents an image in the document.

div

The div tag defines a division. This tag allows grouping several block elements (paragraphs, headings, lists, tables, divisions, etc).

style

It is the element in charge of indicating the style information. Its tags are: style and / style (both mandatory).
